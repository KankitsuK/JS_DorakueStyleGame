フレームワーク無しのJavascript単体でどこまで作れるか試しました。
開発期間は2週間くらい。

その制御でsettimeputを使ったことでなんとなく非同期処理が理解できたような気がします。
また、実際に作っていて技術力の低さを痛感しています。これやるならこの実装方法は微妙だったな……と
後で頭を抱える事が多かったです。でも楽しかったのでまた作りたいです。
